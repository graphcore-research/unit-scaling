unit_scaling.modules.MHSA

class unit_scaling.modules.MHSA(hidden_size: int, heads: int, dropout_p: float = 0.1, constraint: ~typing.Optional[~typing.Callable[[...], float]] = <function gmean>)[source]

A unit-scaled implementation of a multi-head self-attention layer.

Parameters:

  • hidden_size (int) – the hidden dimension size of the input.

  • heads (int) – the number of attention heads.

  • dropout_p (float, optional) – the probability of the post-softmax dropout. Defaults to 0.1.

  • constraint (Optional[Callable[..., float]], optional) – function which takes any number of output/grad-input scales and returns a single ‘constrained’ scale. If None is provided, no constraint will be applied. Defaults to gmean.
