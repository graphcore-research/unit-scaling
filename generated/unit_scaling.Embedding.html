3.1.9. unit_scaling.EmbeddingÔÉÅ

class unit_scaling.Embedding(num_embeddings: int, embedding_dim: int, padding_idx: int | None = None, max_norm: float | None = None, norm_type: float = 2.0, scale_grad_by_freq: bool = False, sparse: bool = False, _weight: Tensor | None = None, _freeze: bool = False, device: Any = None, dtype: Any = None)[source]ÔÉÅ

A unit-scaled lookup table that looks up embeddings in a fixed dictionary and size.

This module is often used to store word embeddings and retrieve them using indices. The input to the module is a list of indices, and the output is the corresponding word embeddings.

Parameters:

  • num_embeddings (int) ‚Äì size of the dictionary of embeddings

  • embedding_dim (int) ‚Äì the size of each embedding vector

  • padding_idx (int?) ‚Äì If specified, the entries at padding_idx do not contribute to the gradient; therefore, the embedding vector at padding_idx is not updated during training, i.e. it remains as a fixed ‚Äúpad‚Äù. For a newly constructed Embedding, the embedding vector at padding_idx will default to all zeros, but can be updated to another value to be used as the padding vector.

  • max_norm (float?) ‚Äì If given, each embedding vector with norm larger than max_norm is renormalized to have norm max_norm.

  • norm_type (float?) ‚Äì The p of the p-norm to compute for the max_norm option. Default 2.

  • scale_grad_by_freq (bool?) ‚Äì [not supported by unit-scaling] If given, this will scale gradients by the inverse of frequency of the words in the mini-batch. Default False.

  • sparse (bool?) ‚Äì [not supported by unit-scaling] If True, gradient w.r.t. weight matrix will be a sparse tensor. See Notes for more details regarding sparse gradients.

weightÔÉÅ

the learnable weights of the module of shape (num_embeddings, embedding_dim) initialized from \(\mathcal{N}(0, 1)\)

Type:

Tensor

Shape:

  • Input: \((*)\), IntTensor or LongTensor of arbitrary shape containing the indices to extract

  • Output: \((*, H)\), where * is the input shape and \(H=\text{embedding\_dim}\)

Examples

>>> # an Embedding module containing 10 tensors of size 3
>>> embedding = nn.Embedding(10, 3)
>>> # a batch of 2 samples of 4 indices each
>>> input = torch.LongTensor([[1, 2, 4, 5], [4, 3, 2, 9]])
>>> # xdoctest: +IGNORE_WANT("non-deterministic")
>>> embedding(input)

         [-0.6431,  0.0748,  0.6969],
         [ 1.4970,  1.3448, -0.9685],
         [-0.3677, -2.7265, -0.1685]],

[ 0.4362, -0.4004, 0.9400], [-0.6431, 0.0748, 0.6969], [ 0.9124, -2.3616, 1.1151]]])

>>> # example with padding_idx
>>> embedding = nn.Embedding(10, 3, padding_idx=0)
>>> input = torch.LongTensor([[0, 2, 0, 5]])

classmethod from_pretrained(embeddings, freeze=True, padding_idx=None, max_norm=None, norm_type=2.0, scale_grad_by_freq=False, sparse=False)[source]ÔÉÅ

Create Embedding instance from given 2-dimensional FloatTensor.

Parameters:

  • embeddings (Tensor) ‚Äì FloatTensor containing weights for the Embedding. First dimension is being passed to Embedding as num_embeddings, second as embedding_dim.

  • freeze (bool, optional) ‚Äì If True, the tensor does not get updated in the learning process. Equivalent to embedding.weight.requires_grad = False. Default: True

  • padding_idx (int, optional) ‚Äì If specified, the entries at padding_idx do not contribute to the gradient; therefore, the embedding vector at padding_idx is not updated during training, i.e. it remains as a fixed ‚Äúpad‚Äù.

  • max_norm (float, optional) ‚Äì See module initialization documentation.

  • norm_type (float, optional) ‚Äì See module initialization documentation. Default 2.

  • scale_grad_by_freq (bool, optional) ‚Äì See module initialization documentation. Default False.

  • sparse (bool, optional) ‚Äì See module initialization documentation.

Examples:

>>> # FloatTensor containing pretrained weights
>>> weight = torch.FloatTensor([[1, 2.3, 3], [4, 5.1, 6.3]])
>>> embedding = nn.Embedding.from_pretrained(weight)
>>> # Get embeddings for index 1
>>> input = torch.LongTensor([1])
>>> # xdoctest: +IGNORE_WANT("non-deterministic")
>>> embedding(input)
tensor([[ 4.0000,  5.1000,  6.3000]])
