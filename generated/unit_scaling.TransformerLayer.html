class unit_scaling.TransformerLayer(hidden_size: int, heads: int, mhsa_tau: float, mlp_tau: float, is_causal: bool, dropout_p: float = 0.0)[source]ÔÉÅ

A unit-scaled implementation of a PreNorm (see https://arxiv.org/abs/2002.04745) transformer layer.

Warning: using constraint=None here will likely give incorrect gradients.

Parameters:

  • hidden_size (int) ‚Äì the hidden dimension size of the input.

  • heads (int) ‚Äì the number of attention heads.

  • mhsa_tau (float) ‚Äì the weighting of the multi-head-self-attention branch relative to the skip connection.

  • mlp_tau (float) ‚Äì the weighting of the MLP branch relative to the skip connection.

  • is_causal (bool) ‚Äì causal masking (for non-padded sequences).

  • dropout_p (float, optional) ‚Äì the probability of residual and post-softmax dropout.
