4.5.3. unit_scaling.functional.dropout

unit_scaling.functional.dropout(input: Tensor, p: float = 0.5, training: bool = True, inplace: bool = False) → Tensor[source]

Applies a unit-scaled dropout function. tensor with probability p using samples from a Bernoulli distribution.

See Dropout for details.

Parameters:

  • p – probability of an element to be zeroed. Default: 0.5

  • training – apply dropout if is True. Default: True

  • inplace – [not supported by unit-scaling] If set to True, will do this operation in-place. Default: False
